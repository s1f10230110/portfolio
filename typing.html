タイピングゲーム

GitHub

概要

React を使用したタイピングゲーム。時間制限付きで、短時間でも手軽にプレイできる。シンプルで見やすい UI により、集中してタイピング練習ができる。

ターゲット

タイピングを手軽に練習したい人。

課題

タイピングは練習しないと上達しないが、長時間取り組むのは面倒。短時間でゲーム感覚で練習できる環境が求められている。

目的

短時間で手軽にタイピング練習ができる仕組みを提供し、継続的に練習できるようにする。

使用技術

フレームワーク：React
JavaScript / HTML / CSS

制作情報

制作時期：2025年度 春学期
制作期間：1週間
