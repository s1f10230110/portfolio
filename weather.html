スカイツリー・浅草観光サイト

概要

スカイツリータウンと浅草を家族向けに紹介する観光Webサイト。マップ上に観光スポットをアイコンで表示し、詳細情報や　3日間の天気予報を確認できる。レスポンシブ対応で、PCでもスマホでも使いやすく、観光計画をサポートする。

ターゲット

家族で観光を計画している人。事前にしっかりと下調べをしてから出かけたいと考えている層を想定する。

課題

観光情報が分散しており、スポットや天気、マップなどを個別に調べる手間がある。限られた時間の中で、無理なく効率よく回れる計画を立てるのが難しい。

目的

スカイツリータウンと浅草への関心を高め、実際に観光してもらうこと。家族向けに観光情報を一か所にまとめ、計画から当日までをスムーズにサポートする。

使用技術

言語：HTML / CSS / JavaScript
ライブラリ・ツール：Bootstrap
